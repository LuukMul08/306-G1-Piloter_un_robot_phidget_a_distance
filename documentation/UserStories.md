# User Stories

## Connexion et statut

En tant qu’utilisateur, je veux voir si le robot est connecté et prêt.
Je peux savoir si tout fonctionne ou s’il y a un problème.

## Déplacement du robot

En tant qu’utilisateur, je veux avancer, reculer et tourner le robot.
Je peux le déplacer dans toutes les directions.

## Arrêt d’urgence

En tant qu’utilisateur, je veux un bouton pour arrêter le robot immédiatement.
Je peux stopper le robot si nécessaire pour la sécurité.

## Vitesse réglable

En tant qu’utilisateur, je veux changer la vitesse du robot : lent, moyen, rapide.
Je peux adapter la vitesse selon la situation.

## Interface simple

En tant qu’utilisateur, je veux une interface claire et facile à utiliser.
Je peux piloter le robot sans formation longue.

## Contrôle avec Xbox

En tant qu’utilisateur, je veux piloter le robot avec un contrôleur Xbox.
Je peux utiliser les sticks et les boutons pour contrôler le robot.

## Contrôle avec clavier

En tant qu’utilisateur, je veux piloter le robot avec le clavier si je n’ai pas de contrôleur.
Je peux utiliser les touches fléchées ou WASD pour avancer, reculer et tourner.
